# Glossar

| Begriff | Definition |
|---|---|
| Baseline | Versionierter Referenzlauf mit festem Rechts-, Daten-, Preis- und Bevölkerungsstand. |
| Policy Lever | Veränderbarer politischer Parameter, etwa Steuersatz oder Ausgabenbudget. |
| Direkte Wirkung | Unmittelbare rechnerische Änderung ohne Verhaltensanpassung. |
| Indirekte Wirkung | Folge über mindestens einen vermittelnden Mechanismus. |
| Synthetischer Haushalt | Gewichteter Modellhaushalt, der ähnliche reale Haushalte repräsentiert. |
| Claim | Aus einer Quelle abgeleitete, prüfbare Aussage. |
| Effect Edge | Gerichteter Zusammenhang im Wirkungsgraphen. |
| Provenienz | Nachweis von Daten, Annahmen, Formeln und Versionen eines Ergebnisses. |
| Scheingenauigkeit | Exakte Darstellung trotz unzureichender Daten- oder Modellpräzision. |
| Kohorte | Personen eines gemeinsamen Geburts- oder Altersjahrgangs, der in der Projektion jährlich altert. |
| Kohorten-Komponenten-Modell | Aggregiertes Modell, das Alterung, Geburten, Sterbefälle, Zu- und Fortzüge kombiniert. |
| Fertilitätsrate | Maß der Geburtenhäufigkeit; eine politische Leistung erzeugt daraus nicht automatisch eine feste zusätzliche Geburtenzahl. |
| Wanderungssaldo | Zuzüge minus Fortzüge im betrachteten Jahr. |
| Schutzsuchende | Personengruppe mit schutz- oder asylbezogenem Status; nicht gleichbedeutend mit Erwerbsberechtigung oder Beschäftigung. |
| Asylbewerbende | Personen im Asylverfahren; der Status allein beschreibt weder Arbeitsmarktzugang noch spätere Beschäftigung. |
| Erwerbsberechtigung | Rechtliche Voraussetzung für eine Beschäftigung; keine Beschäftigungsgarantie. |
| Erwerbsperson | Person, die unter der Beteiligungsannahme dem Arbeitsmarkt zur Verfügung steht. |
| Beschäftigungsquote | Anteil der Erwerbspersonen, für den im Modell Beschäftigung angenommen wird. |
| Bevölkerung im Erwerbsalter | Personen zwischen der unteren Altersgrenze und der im Szenario gesetzten Rentenaltersgrenze. |
| Erwerbspotenzial | Demografisch und verhaltensbezogen abgeleitete Arbeitsmarktstufe vor der Beschäftigung. |
| Vollzeitäquivalent | Beschäftigungsvolumen, das Teilzeit und Arbeitszeitfaktoren auf eine Vollzeitgröße umrechnet. |
| Beitragszahlende | Im Basismodell Beschäftigte mit beitragspflichtigem Arbeitsvolumen. |
| Rentenbeziehende | Ab Rentenzugangsalter geschätzte Personen mit einer Bezugsquote; nicht die gesamte ältere Bevölkerung. |
| Altenquotient | Verhältnis einer älteren Bevölkerungsgruppe zur Bevölkerung im Erwerbsalter; kein Finanzierungssaldo. |
| Vereinfachter Finanzierungssaldo | Beiträge plus getrennte weitere Einnahmen und Bundeszuschuss minus Renten- und Verwaltungsausgaben. |
